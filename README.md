# Portable App Installer

Графический установщик программ для «чистого» ПК: добавляете ваши установщики (по одному или из папки), выбираете нужные, и запускаете установку параллельно в несколько потоков.

## Что уже реализовано

- Добавление установщиков через интерфейс.
- Массовый импорт всех установщиков из выбранной папки.
- Сохранение списка программ в `installer_catalog.json`.
- Выборочная установка или установка всех элементов.
- Параллельный запуск установок (одновременно несколько программ).
- Общий прогресс и статус по каждой программе.
- На Linux/macOS: ввод sudo-пароля один раз в начале через `sudo -v`.

## Поддерживаемые типы файлов

- Windows: `.exe`, `.msi`, `.msix`
- Linux: `.deb`, `.rpm`, `.sh`
- macOS: `.pkg`

> Для «тихой» установки (`silent`) у разных программ разные ключи. Приложение подставляет базовые, но в интерфейсе можно изменить аргументы для каждой программы.

## Запуск

```bash
python3 app_installer_gui.py
```

## Как использовать

1. Нажмите **«Добавить установщик»** для одиночного добавления.
2. Или нажмите **«Импортировать папку»**, чтобы подтянуть все установщики из каталога.
3. При необходимости откройте строку двойным кликом и измените имя/путь/аргументы.
4. Нажмите **«Установить выбранные»** или **«Установить все»**.
5. На Linux/macOS пароль администратора вводится один раз перед стартом.

## Файл каталога

Список сохраняется в `installer_catalog.json` рядом со скриптом.
